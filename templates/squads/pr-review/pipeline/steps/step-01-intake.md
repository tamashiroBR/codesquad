# Step 01 — PR Alvo (checkpoint)

Ask the user for, then save `output/intake.md`:

1. **The PR** — a GitHub PR URL or `owner/repo#number`. The `github` skill reads the diff,
   description, and linked issue.
2. **Review focus (optional)** — anything to weight (security, performance, a specific module).

Confirm which PR is under review in one sentence. Checkpoint: do not proceed until the user confirms.
