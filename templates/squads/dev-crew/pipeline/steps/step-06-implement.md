# Step 06 — Implement (agent: coder)

Dispatch **Caio Código** with `tasks/implement.md`. Input: approved spec + design + conventions.
Creates a topic branch and implements strictly within the approved scope, in separate logical
commits. Output: the changeset + `output/implementation-notes.md`. If the design proves
unworkable, the coder stops and reports rather than improvising.

> This step is also the return target for the `green-suite` and `review-verdict` gates.
